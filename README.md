# Water Potability Prediction using Machine Learning

## Overview

This project predicts whether water is safe for drinking based on water quality parameters using Machine Learning and Streamlit.

## Features

* Water Potability Prediction
* Confidence Score
* User-Friendly Interface
* Real-Time Prediction

## Input Parameters

* pH
* Hardness
* Solids
* Chloramines
* Sulfate
* Conductivity
* Organic Carbon
* Trihalomethanes
* Turbidity

## Technologies Used

* Python
* Streamlit
* Scikit-Learn
* Pandas
* NumPy

## Run Locally

pip install -r requirements.txt

streamlit run app.py
